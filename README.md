# Number Guessing Game (Python)

A fun and simple Python game where the computer randomly selects a number between 1 and 50, and you have 5 attempts to guess it correctly. Itâ€™s a perfect beginner-friendly project to practice Python basics.

## How to Play
1. Run the game: `python guess.py`
2. Enter your guess.
3. The game will tell you if your guess is too high, too low, or correct.
4. After 5 incorrect attempts, the correct number is revealed.

## Features
- Random number generation using Python's `random` module
- 5 guessing attempts
- Clear and user-friendly messages
- Beginner-friendly logic
